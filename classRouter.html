Main entry point for determining how an HTTP request should be handled.

The Router decides the appropriate route type based on request attributes, location configuration, and server rules. It checks for:

  • Parser errors or invalid URIs
  • Path traversal attempts
  • Redirects
  • CGI execution
  • File uploads
  • Autoindex directories
  • Static file serving
  • DELETE operations

On success, sets the correct RouteType and HTTP response status.
