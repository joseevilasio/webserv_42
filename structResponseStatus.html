Detailed Description

Contains standard HTTP status codes grouped by category.

This struct defines an enumeration of commonly used HTTP response status codes, based on RFC 2616 and related specifications.

Categories:

  • 1xx Informational: Request received, continuing process.
  • 2xx Successful: Request successfully received and understood.
  • 3xx Redirection: Further action must be taken to complete the request.
  • 4xx Client Error: The request contains bad syntax or cannot be fulfilled.
  • 5xx Server Error: The server failed to fulfill a valid request.
